# Teamverwaltung — Hilfe für Mitglieder und Admins

Diese Dokumentation richtet sich an **Mitglieder und Administrator:innen**
eines Vereins- bzw. Teams, das mit Teamverwaltung verwaltet wird — nicht an
Entwickler:innen. Für die technische Dokumentation siehe
[`CLAUDE.md`](https://github.com/yoadey/team-manager/blob/main/CLAUDE.md) und
[`docs/operations.md`](https://github.com/yoadey/team-manager/blob/main/docs/operations.md).

Die Anleitungen sind aktuell **nur auf Deutsch** verfügbar. Eine englische
Version ist als Folgeschritt geplant.

## Inhalt

### Einstieg

- [Erste Schritte](./erste-schritte.md) — Einladung, erster Login, Team
  anlegen oder beitreten
- [Rollen & Rechte](./rollen-und-rechte.md) — was Rollen sind und was
  „Lesen"/„Schreiben"/„—" bedeuten

### Funktionen

- [Termine](./termine.md) — Liste, Kalender, Zu-/Absagen, Abwesenheiten,
  Kalender-Export
- [Mitglieder](./mitglieder.md) — Profile, Rollen zuweisen
- [Finanzen](./finanzen.md) — Umsätze, Strafen, Beiträge
- [Neuigkeiten](./news.md) — Team-News lesen und verfassen
- [Umfragen](./umfragen.md) — Umfragen erstellen und abstimmen
- [Team-Einstellungen](./team-einstellungen.md) — Einladungen, Team-Profil,
  Rollenverwaltung, Team wechseln
- [Statistik](./statistik.md) — Anwesenheitsauswertung

### Konto & Datenschutz

- [Daten & Datenschutz](./daten-und-datenschutz.md) — eigene Daten
  exportieren, Konto löschen
