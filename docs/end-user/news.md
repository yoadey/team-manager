# Neuigkeiten

Im Bereich *Neuigkeiten* teilt das Team Ankündigungen und Aktuelles.

- Jede Neuigkeit hat einen Titel und einen Text.
- Mit „Schreiben" im Modul *News* lassen sich Neuigkeiten verfassen,
  bearbeiten, **oben anpinnen** (damit sie an erster Stelle bleibt) und
  löschen — Löschen entfernt den Eintrag dauerhaft.
- Ohne Schreibrecht könnt ihr Neuigkeiten nur lesen.

Die neuesten Neuigkeiten erscheinen zusätzlich auf der
[Startseite](./erste-schritte.md#die-startseite).
