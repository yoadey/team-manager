# Umfragen

Im Bereich *Umfragen* lassen sich Meinungsbilder im Team einholen.

## Umfrage erstellen (mit Schreibrecht)

Mit „Schreiben" im Modul *Umfragen* könnt ihr eine neue Umfrage anlegen:

- eine Frage,
- **zwei bis vier Antwortoptionen**,
- optional **Mehrfachauswahl** (mehr als eine Option wählbar) statt
  Einfachauswahl,
- optional **Anonym** — dann wird beim Ergebnis nicht angezeigt, wer wie
  abgestimmt hat, nur die Stimmenzahl je Option.

## Abstimmen und Ergebnisse

Alle Mitglieder mit Zugriff auf das Modul können abstimmen. Die
Ergebnisansicht zeigt die Stimmenzahl je Option; bei anonymen Umfragen ist
das entsprechend gekennzeichnet.

## Löschen

Eine Umfrage löschen (mit „Schreiben") entfernt sie samt aller bisherigen
Abstimmungsergebnisse **dauerhaft**.
